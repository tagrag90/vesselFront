# 에디터 페이지 디자인 및 개선안 보고서
**Studio_bada Vessel Editor Improvement Plan v1.0**

---

## 📋 현재 상태 분석

### 현재 구조
1. **BlogEditor 컴포넌트**
   - 단일 컬럼 레이아웃 (`max-w-[900px]`)
   - 배너 이미지 → 제목 → 에디터 (수직 배치)
   - 상단 네비게이션 바 (로고, 제목, Publish/Save Draft 버튼)

2. **PublishForm 컴포넌트**
   - 2칼럼 그리드 레이아웃 (`lg:grid-cols-2`)
   - 왼쪽: 미리보기 (배너, 제목, 설명)
   - 오른쪽: 메타데이터 입력 폼 (제목, 설명, 태그)

3. **문제점**
   - 에디터와 메타데이터 입력이 분리되어 있어 페이지 전환이 필요
   - 좌우 여백 활용도가 낮음 (현재 900px 고정)
   - 모바일/태블릿 대응 부족
   - 에디터 툴바 접근성이 떨어짐

## 🎨 개선안 1: 에디터 디자인 개선

### 1.1 네비게이션 바 개선
```
현재:
- 상단 고정, 로고/제목/버튼 가로 배치
- 배경: 흰색, 텍스트: 검은색

개선안:
- 상단 고정 네비게이션 바 유지
- 제목 입력을 네비게이션 바로 이동 (선택적)
- 버튼 스타일: 아이콘 + 텍스트 조합
- 자동 저장 상태 표시 (Last saved 2 mins ago)
```

### 1.2 에디터 영역 개선
```
현재:
- 단순 텍스트 에리어 + EditorJS
- 배너 이미지가 상단에 큰 공간 차지

개선안:
- EditorJS 툴바 개선 (플로팅 툴바)
- 인라인 배너 업로드 (에디터 내부에서 처리)
- 블록별 미리보기 개선
- 실시간 자동저장 인디케이터
- 에디터 폭 최적화 (700px - 가독성 고려)
```

### 1.3 타이포그래피 개선
```
현재:
- text-4xl 제목, text-xl 본문

개선안:
- 제목: text-5xl (50px, 본문 페이지와 일치)
- 본문: text-xl (16px base, 20px 데스크톱)
- 행간: 1.8 (본문), 1.4 (제목)
- 폰트: Pretendard (Studio_bada 규칙 준수)
```

### 2.2 반응형 레이아웃 전략

#### 데스크톱 (≥1280px)
```
메인 컨테이너: max-w-[1400px] mx-auto
- 왼쪽 컬럼 (에디터): flex-1, max-w-[800px]
- 오른쪽 컬럼 (사이드바): w-[400px], sticky top-24
- 간격: 48px
```

#### 태블릿 (768px ~ 1279px)
```
- 2칼럼 유지
- 왼쪽 컬럼: flex-1
- 오른쪽 컬럼: w-[320px]
- 간격: 32px
```

#### 모바일 (<768px)
```
- 단일 컬럼 (세로 스택)
- 에디터 → 사이드바 순서
- 사이드바 고정 해제 (sticky 제거)
```

### 2.3 사이드바 컴포넌트 구성

```
1. 배너 이미지 섹션
   - 드래그 앤 드롭 업로드
   - 이미지 미리보기 (16:9 비율)
   - 변경/삭제 버튼

2. 제목 입력
   - 실시간 동기화 (에디터 제목과)
   - 문자 수 카운터 (선택적)

3. 설명 입력
   - Textarea (200자 제한)
   - 실시간 카운터
   - 미리보기 텍스트 표시

4. 태그 입력
   - 인풋 필드 + 엔터/쉼표로 추가
   - 태그 칩 표시
   - 삭제 가능한 태그
   - 최대 10개 제한 표시

5. 미리보기 카드
   - 실시간 업데이트
   - 블로그 리스트 페이지 카드 스타일
   - 호버 효과
```

### 2.4 스크롤 및 네비게이션

```
- 왼쪽 에디터: 독립 스크롤 (필요 시)
- 오른쪽 사이드바: sticky position (top: 96px)
- 스크롤 시 사이드바는 고정, 에디터만 스크롤
- 모바일: 사이드바 sticky 해제, 자연스러운 스크롤
```

## ✨ 개선안 3: 에디터 기능 개선

### 3.1 EditorJS 개선
```
1. 플로팅 툴바
   - 텍스트 선택 시 나타나는 툴바
   - 인라인 포맷팅 옵션

2. 블록 타입 개선
   - 이미지 블록: 캡션 추가 기능
   - 인용구: 스타일 개선
   - 코드 블록: 구문 강조

3. 자동 저장
   - 30초마다 자동 저장
   - 저장 상태 표시 (저장됨/저장 중)
   - 충돌 감지 및 해결
```

### 3.2 실시간 미리보기
```
- 사이드바 미리보기 카드 업데이트
- 제목/설명 변경 시 즉시 반영
- 배너 이미지 변경 시 즉시 반영
- 태그 추가/삭제 시 즉시 반영
```

### 3.3 키보드 단축키
```
- Cmd/Ctrl + S: 저장 (드래프트)
- Cmd/Ctrl + Enter: 발행
- Cmd/Ctrl + K: 블록 삽입
- Cmd/Ctrl + B: 볼드
- Cmd/Ctrl + I: 이탤릭
```
