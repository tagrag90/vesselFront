# 히어로 배너 개선안

## 현재 상태 분석

### 현재 구조
- Swiper 슬라이드 (Fade 효과)
- 이미지와 텍스트 분리 (이미지 위, 텍스트 아래)
- 중앙 정렬 레이아웃
- 배경 이미지 + 별도 콘텐츠 영역
- 인디케이터와 네비게이션 버튼

### 주요 문제점
1. **시각적 임팩트 부족**: 이미지와 텍스트가 분리되어 임팩트가 약함
2. **레이아웃 비효율**: 이미지 영역만 높고 텍스트가 별도 영역에 있어 공간 활용이 비효율적
3. **타이포그래피**: 제목 크기와 계층 구조 개선 필요
4. **모바일 경험**: 텍스트 영역이 크게 줄어들어 가독성 저하
5. **인터랙션**: 버튼 텍스트가 영어("Read Article")로 되어 있음

---

## 개선안 제안

### 옵션 A: 풀스크린 오버레이 스타일 (추천)

**개념**: 이미지 전체에 텍스트 오버레이를 배치하여 임팩트 강화

**레이아웃**:
- 전체 화면 높이 (최소 60vh, 데스크톱 70vh)
- 배경 이미지에 그라데이션 오버레이 적용
- 텍스트를 이미지 중앙 또는 하단 좌측에 배치
- 반투명 배경 패널로 텍스트 가독성 확보

**장점**:
- ✅ 강한 시각적 임팩트
- ✅ 공간 효율적
- ✅ 모던하고 세련된 디자인
- ✅ 모바일에서도 텍스트 가독성 유지

**구현 포인트**:
- 그라데이션: `linear-gradient(to top, rgba(0,0,0,0.8) 0%, rgba(0,0,0,0.4) 50%, transparent 100%)`
- 텍스트 영역: 좌측 하단 또는 중앙 배치
- 배경 패널: `bg-black/60 backdrop-blur-sm` 또는 완전 투명
- 버튼: "Read Article" → "읽기" 또는 "자세히 보기"

---

### 옵션 B: 좌우 분할 레이아웃

**개념**: 이미지(좌측 60%)와 텍스트 영역(우측 40%) 분할

**레이아웃**:
- 데스크톱: `grid-cols-[60%_40%]` 또는 `flex`
- 모바일: 세로 배치
- 텍스트 영역에 배경색 또는 그라데이션 적용

**장점**:
- ✅ 정보 전달이 명확함
- ✅ 텍스트 가독성 최대화
- ✅ 프로페셔널한 느낌

**단점**:
- ❌ 화면 공간 많이 차지
- ❌ 모바일에서 비효율적

---

### 옵션 C: 카드 스타일 (Minimal)

**개념**: 작은 카드 형태의 히어로 배너

**레이아웃**:
- 중앙 정렬된 카드 (max-w-4xl)
- 이미지와 텍스트가 한 카드 안에
- 그림자와 호버 효과

**장점**:
- ✅ 미니멀하고 깔끔
- ✅ 컨텐츠에 집중
- ✅ 부드러운 느낌

**단점**:
- ❌ 임팩트가 상대적으로 약함

## 상세 개선사항 (옵션 A 기준)

### 1. 레이아웃 변경
```jsx
// 현재: 이미지 영역 + 별도 텍스트 영역
<div>
  <img />
  <div>텍스트</div>
</div>

// 개선: 이미지 위 오버레이
<div className="relative">
  <img />
  <div className="absolute inset-0 gradient-overlay" />
  <div className="absolute bottom-0 left-0 right-0 p-8 md:p-12">
    {/* 텍스트 콘텐츠 */}
  </div>
</div>
```

### 2. 타이포그래피 개선
- 제목: `text-4xl md:text-6xl lg:text-7xl` (더 크게)
- 설명: `text-base md:text-lg lg:text-xl` (더 크게)
- 간격: `mb-4 md:mb-6` (더 여유있게)

### 3. 그라데이션 오버레이
```css
background: linear-gradient(
  to top,
  rgba(0, 0, 0, 0.85) 0%,
  rgba(0, 0, 0, 0.5) 40%,
  rgba(0, 0, 0, 0.2) 70%,
  transparent 100%
);
```

### 4. 버튼 개선
- 텍스트: "Read Article" → "자세히 보기"
- 스타일: `bg-[#00DD89] text-black` 또는 `bg-white text-black`
- 크기: `px-8 py-4 text-base md:text-lg`
- 호버: `hover:bg-[#00DD89]/90` 또는 `hover:bg-white/90`

### 5. 작성자 정보 스타일
- 아바타: `w-12 h-12 border-2 border-white/50`
- 텍스트: `text-white` (배경이 어두우므로)
- 배경: 투명 또는 반투명 패널

### 6. 인디케이터 개선
- 위치: 텍스트 영역 내부 또는 하단 중앙
- 스타일: 더 큰 점 (w-2 h-2 → w-3 h-3)
- 색상: `bg-white` (어두운 배경 위)

### 7. 네비게이션 버튼
- 위치: 좌우 양끝
- 스타일: `bg-white/20 hover:bg-white/30`
- 크기: 더 크게 (w-14 h-14)
- 아이콘: 화살표 또는 chevron

### 8. 모바일 최적화
- 높이: `min-h-[50vh] md:min-h-[70vh]`
- 패딩: `p-6 md:p-12`
- 폰트 크기: 반응형으로 조정
- 버튼: 전체 너비 또는 중앙 정렬

### 9. 애니메이션
- 페이드 인: 텍스트 요소에 `fade-in` 효과
- 스무스 트랜지션: 슬라이드 전환 시 부드러운 효과
- 호버 효과: 버튼과 인디케이터에 미세한 애니메이션

### 10. 태그 스타일
- 이미 적용된 StudioBadge 스타일 유지
- 위치: 텍스트 영역 상단
- 가시성: 배경이 어두우므로 현재 스타일 유지
